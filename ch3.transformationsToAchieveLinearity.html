Transformations to Achieve Linerarity

    Non-linear data can be transformed in such a way that the transformed data can be modeled by a line
    If a variable grows exponentially, its logarithm grows linearly.
    To transform an exponential function into a linear function, take the ln of the y values
      The new LSRL will be lnŷ = a + bx
    Transformation Process
    1. Plot (x,y), make a scatterplot AND a residual plot to determine if the graph is linear.
        LSRL is ŷ = a + bx
        Example
        IF NOT
    2. Plot (x, lny), make a scatterplot AND a residual plot to determine if the graph is linear.
        LSRL is lnŷ = a + bx
        Example
        IF NOT
    3. Plot (lnx, lny), make a scatterplot AND a residual plot to determine if the graph is linear.
        LSRL is lnŷ = a + blnx
        Example
    To achieve linearity from a power model y = axp, take the logarithm of both variables
