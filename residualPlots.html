Residual Plots

    The difference between the actual and predicted y value
    How to Assess if a Linear Relation Exists
    1. Make a scatterplot: Does it look linear?
    2. Make a residual plot
    Patterns in a residual point
    1. if it's curved, then the relationship is not linear
    2. if there is an increasing/decreasing spread, then the predicted y will be less accurate
    3. inviduals with large residuals are outliers
    4. individual points that are extreme in the x direction are potentially influential points
