Vocabulary

  1. Explanatory Variable
      2. Independent Variable, x
  2. Response Variable
      3. Dependent Variable, y
  3. Correlation Coefficient (r)
      4. Tells us the strength and direction of the linear relationship between the variables
      1. Strong if -0.8 &le r &le -1 or 0.8 &le r &le 1
      2. Moderate if -0.5 &le r &le -0.8 or 0.5 &le r &le 0.8
      3. Weak if -0.5 &le r &le 0.5
  4. Coefficient of Determination (r2)
      5. The percent of the variation in y that can be explained by the linear relationship with x
  5. Least Squares Regression Line (LSRL)
      6. A regression line is a straight line that describes how a response variable changes as an explanatory variable changes. We use a regression line to predict the value of y for a given x
  6. LSRL Equation
      7. ŷ = a + bx
      1. a represents the Y-Intercept
      2. b represents the Slope
