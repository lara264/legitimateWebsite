     Vocabulary

    1. Correlation Coefficient (r)
        2. Tells us the strength and direction of the linear relationship between the variables
        1. Strong if -0.8 &le r &le -1 or 0.8 &le r &le 1
        2. Moderate if -0.5 &le r &le -0.8 or 0.5 &le r &le 0.8
        3. Weak if -0.5 &le r &le 0.5
    2. Coefficient of Determination (r2)
        3. The percent of the variation in y that can be explained by the linear relationship with x
    3. Least Squares Regression Line (LSRL)
        4. A regression line is a straight line that describes how a response variable changes as an explanatory variable changes. We use a regression line to predict the value of y for a given x
    4. LSRL Equation
